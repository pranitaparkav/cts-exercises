Weekly Bulletin

Downtown Project Approved

The city council has approved the downtown pedestrian corridor plans. The construction will widen sidewalks and introduce dedicated bicycle paths along Main Street. Detours will be marked, and the project is expected to finish in six months.

Library Coding Program

A free programming program for teenagers will start next month at the library. Laptops are provided, and mentors from local firms will teach HTML and CSS basics. Register at the information desk.
